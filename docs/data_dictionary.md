# Data Dictionary

## `data/synthetic/hourly_energy_profile.csv`

| Column | Meaning | Unit |
|---|---|---|
| `hour` | Hour of the synthetic day | 1–24 |
| `demand_mwh` | Hourly electricity demand | MWh |
| `solar_mwh` | Hourly solar generation | MWh |
| `wind_mwh` | Hourly wind generation | MWh |
| `thermal_mwh` | Dispatchable thermal generation available | MWh |
| `emissions_factor_tco2_per_mwh` | Synthetic thermal emissions factor | metric tons CO2 per MWh |
