# Engineering Use Notes

This repository helps explain energy systems modeling, but it does not replace:

- Utility-grade planning models
- Protection studies
- Interconnection studies
- NERC, IEEE, IEC, ASME, ISO, or jurisdiction-specific standards
- Professional engineering review
- Safety analysis
- Real-time grid operations tools
- Certified emissions or procurement accounting

Use the examples for education, transparent reasoning, reproducible analysis, and article support.
